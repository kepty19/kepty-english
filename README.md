# kepty-english

#### バージョン: Pre-Alpha 0.0.2.4

このプロジェクトは現在プレアルファ段階です。  
機能は開発途中であり、バグや不具合が発生する可能性があります。  
MVP開発スピードを重視し、パブリックリポジトリとして運用しています。


## バージョン管理方針

バージョンは x.y.z 形式で管理するとします。

- x: メジャーリリース
- y: 機能追加・改善リリース
- z: バグ修正リリース
