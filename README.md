# kepty-english

#### バージョン: Alpha 0.0.3.2

このプロジェクトは現在アルファ段階です。  
コア機能は開発されて、ベータ版に上段することが決まりました。  
MVP開発はこれにて終了いたします。このレポはアーカイブすることにします。

これから新しいレポに移行し、そこにて開発を進めます。なお、親レポである　[Kepty-english](https://github.com/kepty19/kepty-english)は公開を続けます。

## バージョン管理方針

バージョンは x.y.z 形式で管理するとします。

- x: メジャーリリース
- y: 機能追加・改善リリース
- z: バグ修正リリース
